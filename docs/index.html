AlzGraph

Building Generalists for Evidence-Intensive Alzheimer's Disease Reasoning in the Wild

AlzGraph couples AlzKG, a five-layer Alzheimer's knowledge graph over genes, ATN biomarkers, disease stages, treatments, and outcomes, with AlzBench, a five-task benchmark, and a Graph-RAG retriever that serializes literature-weighted, multi-hop reasoning paths for language models.

Quick Start

git clone https://github.com/lihuiliullh/AlzGraph.git
cd AlzGraph
pip install -r requirements.txt

# Build the released seed AlzKG (real statistics)
python scripts/build_seed_kg.py

# Graph-RAG evaluation on Task 1
export OPENROUTER_API_KEY="your_key"
python tasks/t1_clinical_decision_accuracy.py \
  --dataset data/alzbench/t1/mcq.json \
  --triplets data/alzkg/triplets.json \
  --model openai/gpt-4o --mode graph_rag
